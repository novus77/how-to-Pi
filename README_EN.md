# how-to-Pi

This repository is a Chinese tutorial project for "Pi Agent Core Principles and Practice".

It explains Pi as an extensible terminal coding agent harness: agent loop, tool calling, session trees, context compaction, extensions, skills, packages, SDK usage, RPC mode, and JSON event streams.

Start with the Chinese documentation:

- [Docsify home](docs/README.md)
- [Preface](docs/Preface.md)
- [Official Pi docs](https://pi.dev/docs/latest)
- [Pi GitHub repository](https://github.com/earendil-works/pi)

Local preview:

```bash
npx docsify-cli serve docs
```

Verification:

```bash
node scripts/verify-docs.mjs
```
