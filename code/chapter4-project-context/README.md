# Chapter 6 Project Context Example

This directory contains a compact `AGENTS.md` example for a Pi-enabled project.

Copy it into a test repository root, then run:

```bash
pi
```

Ask:

```text
Read AGENTS.md and summarize the constraints you must follow before editing files.
```

Expected behavior:

- Pi should mention Chinese prose and English code.
- Pi should mention non-destructive git rules.
- Pi should mention verification before completion.
- Pi should avoid claiming commands were run unless they were actually run.
