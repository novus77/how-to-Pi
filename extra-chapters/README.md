# Extra Chapters

This directory is for optional chapters that should not interrupt the main learning path.

Good candidates:

- Team-specific Pi settings.
- Provider comparison notes.
- Session sharing and anonymization.
- CI integration patterns.
- Case studies from real repositories.

Each extra chapter should include:

- Why it exists.
- Prerequisites.
- Commands or examples.
- Risks and limitations.
- Links back to the main chapters.
